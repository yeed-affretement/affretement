ℹ️ Comment ça marche ?

1. Les affréteurs répondent à vos emails Outlook de consultation.

2. Leurs réponses arrivent dans la boîte mail dédiée (affretement@…).

3. YEED Affrètement lit les emails via Microsoft Graph API, détecte la consultation liée (par référence CONS-YYYYMM-NNN dans le sujet) et extrait le prix, délai et disponibilité.

4. Vous validez les réponses pré-remplies avant import.

📖 Guide de configuration Azure AD — Microsoft Graph API

⏱️ Durée estimée : 5 à 10 minutes. Opération unique, à faire une seule fois.
1
Accéder au portail Azure
Rendez-vous sur portal.azure.com et connectez-vous avec votre compte Microsoft 365 administrateur (ou demandez à votre admin IT).
2
Créer une inscription d'application
Dans la barre de recherche, tapez App registrations → cliquez sur ➕ New registration.
Remplissez :
  • Name : YEED Affrètement
  • Supported account types : Accounts in this organizational directory only
  • Redirect URI : Single-page application (SPA) → http://localhost:8080
Cliquez Register.
3
Copier les identifiants
Dans la page Overview de votre application, copiez :
  • Application (client) ID → collez dans le champ « Client ID » des paramètres
  • Directory (tenant) ID → collez dans le champ « Tenant ID »
4
Ajouter les permissions API
Allez dans API permissions → Add a permission → Microsoft Graph → Delegated permissions.
Ajoutez ces permissions :
Mail.Read Mail.Read.Shared Mail.Send User.Read Sites.ReadWrite.All Files.ReadWrite.All
Puis cliquez Grant admin consent (bouton vert). Un ✅ vert doit apparaître.
5
Créer / désigner la boîte mail dédiée
Dans Microsoft 365 Admin Center, créez une boîte partagée ou un alias, par exemple affretement@yeedgroup.com.
Assurez-vous que votre compte utilisateur a accès en lecture à cette boîte (Accès délégué ou membre de la boîte partagée).
Demandez aux affréteurs de répondre à cette adresse (ajoutez-la en Reply-To dans vos emails Outlook).
6
Démarrer l'application via start.bat
Le protocole OAuth ne fonctionne pas sur file://. Lancez toujours YEED Affrètement via le fichier start.bat fourni, qui démarre un serveur local sur http://localhost:8080.
@echo off start "" http://localhost:8080/affrètement.html python -m http.server 8080
